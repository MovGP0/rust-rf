Acknowledging rust-rf and scikit-rf
-----------------------------------

For work using rust-rf, cite the rust-rf repository and the upstream scikit-rf
paper on which this port is based:

* `rust-rf <https://github.com/MovGP0/rust-rf>`_
* A. Arsenovic *et al.*, "scikit-rf: An Open Source Python Package for Microwave
  Network Creation, Analysis, and Calibration," *IEEE Microwave Magazine*,
  vol. 23, no. 1, 2022, doi: `10.1109/MMM.2021.3117139
  <https://doi.org/10.1109/MMM.2021.3117139>`_.

The historical publication list on the next page describes work citing the
upstream project; it is retained for provenance and is not a claim that those
publications used rust-rf.
