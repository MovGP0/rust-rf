Constants
---------

Rust module: ``rust_rf::constants``

This module exposes physical constants and unit conversions. Public items, signatures, error conditions, and
source links are generated directly by rustdoc:

.. code-block:: console

   cargo doc --all-features --open

The source implementation is in ``src/constants.rs`` (or the matching module
directory). Rust uses ``snake_case`` functions, typed enums in place of string
options, and ``Result`` for operations that can fail.
