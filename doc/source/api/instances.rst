Instances
---------

Rust module: ``rust_rf::instances``

This module exposes standard media and waveguide instances. Public items, signatures, error conditions, and
source links are generated directly by rustdoc:

.. code-block:: console

   cargo doc --all-features --open

The source implementation is in ``src/instances.rs`` (or the matching module
directory). Rust uses ``snake_case`` functions, typed enums in place of string
options, and ``Result`` for operations that can fail.
