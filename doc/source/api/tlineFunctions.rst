Transmission-line functions
---------------------------

Rust module: ``rust_rf::transmission_line``

This module exposes typed line conversions and loss helpers. Public items, signatures, error conditions, and
source links are generated directly by rustdoc:

.. code-block:: console

   cargo doc --all-features --open

The source implementation is in ``src/transmission_line.rs`` (or the matching module
directory). Rust uses ``snake_case`` functions, typed enums in place of string
options, and ``Result`` for operations that can fail.
